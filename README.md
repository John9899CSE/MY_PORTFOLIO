## Portfolio Link

- Live portfolio / repository: https://john9899cse.github.io/MY_PORTFOLIO/

## Files

- `index.html` — main portfolio page
- `styles.css` — layout, typography, and theme styling
- `script.js` — theme toggle, mobile menu, profile image upload, and footer generation
- `assets/` — portfolio assets such as the profile image
- `resume.txt` / `resume.pdf` — resume content

## Features

- Responsive layout for desktop and mobile
- Light/dark theme toggle
- Mobile navigation menu
- Profile photo upload with localStorage persistence
- Smooth scrolling navigation
- Contact section with direct links

## Usage

1. Open `index.html` in a browser.
2. Use the `☀️` / `🌙` button to switch themes.
3. Click the menu button on smaller screens to reveal navigation links.
4. Upload a profile image using the `Upload your photo` control.

## Notes

- The page uses localStorage to remember the selected theme and uploaded profile image.
- The built-in footer is inserted dynamically by `script.js`.
